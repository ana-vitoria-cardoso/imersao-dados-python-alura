# Imersão Dados Python – Alura

Este projeto consiste em um dashboard interativo desenvolvido em Python, com o objetivo de analisar dados salariais da área de dados ao longo dos últimos anos. A aplicação permite explorar informações como cargos, senioridade, tipo de contrato, tamanho da empresa, modelo de trabalho e localização geográfica.
O projeto foi desenvolvido durante a Imersão Dados com Python da Alura, colocando em prática conceitos de análise de dados, visualização e desenvolvimento de aplicações web.

Tecnologias Utilizadas:
Python
Pandas – manipulação e análise dos dados
Streamlit – criação da aplicação web interativa
Plotly Express – visualização de dados interativa
Git & GitHub – versionamento e compartilhamento do projeto
VS Code – ambiente de desenvolvimento

Fonte dos Dados:
Os dados utilizados são carregados diretamente de um arquivo CSV hospedado no GitHub, contendo informações sobre salários na área de dados, como:
Ano
Cargo
Senioridade
Tipo de contrato
Tamanho da empresa
Modalidade de trabalho (remoto, híbrido ou presencial)
País de residência
Salário anual em USD

Funcionalidades do Dashboard:
Filtrar os dados por:
Ano
Senioridade
Tipo de contrato
Tamanho da empresa
Visualizar métricas principais (KPIs), como: Salário médio, Salário máximo, Total de registros, Cargo mais frequente...
Explorar gráficos interativos:
Top 10 cargos com maior salário médio
Distribuição dos salários anuais
Proporção dos tipos de trabalho
Mapa mundial com o salário médio de Cientistas de Dados por país

Visualizar os dados detalhados em formato de tabela
